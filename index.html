Arduino IoT Hub

Aplicación Android nativa (Kotlin) para controlar y monitorizar una placa Arduino vía Bluetooth. Incluye autenticación con Firebase, registro de eventos en Cloud Firestore, consola serial para envío/recepción de comandos, monitor en tiempo real de sensor LDR y control remoto de LEDs. Interfaz moderna basada en Material Design 3 y flujo completo de usuario (Splash, login, panel).
